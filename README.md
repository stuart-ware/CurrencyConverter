Converter with API for #AluraLatam

### This project needs Gson library to work properly
Made in Java 25. JSON parsing by Gson 2.13.2

## How to use?
Simply select the currency you want to exchange. If you don't find any currency you know in the list,
you can manually type the currency from the available list. You need to know which currency code to use.

The list is the program "Currency List" option.
API for [ExchangeRate](exchangerate-api.com)
